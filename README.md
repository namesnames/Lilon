# be-hackathon-Lilion
Youtube에서 원하는 음악을 검색하여 해당하는 영상의 URL을 복사 붙여넣기하면 Youtube-dl과 FFmpeg을 이용하여 *.mp3 확장자 형태로 변경하여 다운로드 합니다. 
그리고, eyeD3으로 ID3 태그를 Youtube에서 정보들을 파싱하여 설정을 해주는 코드를 작성하겠습니다.
